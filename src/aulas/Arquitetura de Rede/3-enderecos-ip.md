# 🌐 Endereços IP e Comunicação em Rede (Módulo 3)

Este módulo aborda os conceitos fundamentais de endereçamento IP e os principais componentes envolvidos na comunicação na camada de rede.

---

## 📌 Endereços IP

O endereço IP (Internet Protocol) é um identificador único atribuído a cada dispositivo conectado a uma rede. Ele permite a comunicação entre dispositivos.

---

## 🔢 Tipos de Endereço IP

### IPv4

Formato numérico composto por quatro octetos.

**Exemplo:**
192.168.0.1

![IPv4](../Arquitetura%20de%20Rede/imagens/19.webp)

---

### IPv6

Formato hexadecimal mais longo, criado para suportar um maior número de dispositivos.

**Exemplo:**
2001:0db8:85a3::8a2e:0370:7334

![IPv6](../Arquitetura%20de%20Rede/imagens/20.webp)

---

## 🌍 IP Público e Privado

* **IP Público**: utilizado para acesso à internet, fornecido pelo provedor.
* **IP Privado**: utilizado dentro de redes locais (LAN).
![IPs](../Arquitetura%20de%20Rede/imagens/21.webp)

---

## 🧩 Comunicação com IP

A comunicação em rede ocorre por meio da troca de pacotes de dados entre dispositivos, utilizando endereços IP de origem e destino.

---

## 🔌 Componentes da Camada de Rede

### 📦 Pacotes IP

Unidades de dados que trafegam na rede contendo informações de origem e destino.

---

### 🚦 Roteamento

Processo de encaminhamento dos pacotes pela rede até o destino.

---

### 🌉 Gateway

Dispositivo responsável por conectar a rede local à internet.

---

### 🗺️ Tabela de Roteamento

Lista de rotas que orienta o envio dos pacotes na rede.

---

## ⚙️ Funcionamento da Comunicação

1. O dispositivo cria um pacote de dados
2. O pacote recebe um endereço IP de destino
3. O roteador define o melhor caminho
4. O pacote é entregue ao destino
